# SkyCast - Weather App (The Odin Project)

## Project description

A simple client-side weather web application built using HTML, CSS, and JavaScript.

## Features

-   Search location.
-   Show searched location current day weather forecast.

## Technologies

-   **HTML**
-   **CSS**
-   **JavaScript**
-   **Weather API**
-   **Eslint & Prettier**
-   **Webpack**
-   **NPM**
